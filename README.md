## ✨ Возможности

### 📋 Редактор смет
- Разделы: добавить / удалить / свернуть / развернуть
- Строки: наименование, ед. изм., количество, цена работы, цена материала
- Итог строки считается автоматически
- НДС 12% — переключатель
- Копировать итог в буфер обмена

### 🗄️ База цен (90+ позиций)
Встроенная база актуальных рыночных цен Казахстана 2024-2025:

| Категория | Позиций |
|-----------|---------|
| ⚡ Электромонтаж | 20 |
| 🚿 Сантехника | 15 |
| 💨 Вентиляция и кондиционирование | 7 |
| 🏗️ Строительство | 15 |
| 🔧 Общестроительные работы | 10 |
| 📡 Слаботочные системы | 7 |
| 🌿 Наружные работы | 6 |

**Два режима поиска:**
- **База цен** — каталог с поиском, кликни на позицию и она добавится в раздел
- **Авто** — автоматически подбирает цены для уже введённых строк по названию

### 🤖 ИИ-оценка (Claude API)
- Вводишь названия работ → Claude анализирует и проставляет цены
- Учитывает тип объекта: промышленный, вахтовый городок, офис, склад и др.
- Заполняет цены работ, материалов, исправляет единицы измерения
- Добавляет комментарий к каждой строке

### 📄 Вкладка «Смета»
Чистый печатный вид с разбивкой по разделам и итоговой таблицей.

## 🚀 Запуск локально

```bash
# Клонировать репозиторий
git clone https://github.com/ваш-логин/smet-calc.git
cd smet-calc

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
# → http://localhost:5173
```

## 🤖 Настройка ИИ-оценки

ИИ-оценка работает через [Anthropic API](https://console.anthropic.com). Для использования:

1. Зарегистрируйся на [console.anthropic.com](https://console.anthropic.com)
2. Создай API ключ
3. В калькуляторе нажми **✨ ИИ-оценка** на любом разделе
4. Введи API ключ в поле `sk-ant-...`

> Стоимость: ~$0.003 за один запрос (один раздел)
